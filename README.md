# PE_V2X_Reliability

**Language:** English | [Français](README_FR.md)

A reproducible, mechanism-oriented simulation pipeline to evaluate **V2X safety-message reliability and timeliness** under **degraded environments** (urban obstruction, tunnels) and **busy-hour congestion**.

This project focuses on:
- timely PDR vs distance
- tail latency (p95 / p99)
- deadline-aware decomposition: PHY success vs late penalty

---

## Repository content

- `03_sim_A/`: simulation code (Python pipeline + MATLAB plotting scripts)
- `06_report/`: report notebooks (EN/FR) + final figure assets (PDF + PNG previews)

Core experimental dimensions:
- Scenarios: `RefPlus`, `UrbMaskPlus`, `TunnelPlus`
- Regimes (ablation): `NoCong` vs `Cong` (congestion/competition toggle)
- Main enhancement lever: retransmissions `ret = 0 / 1 / 2`

> Note: folders like `__pycache__/` are auto-generated by Python and are not part of the repo.

## Quick start (Python)

Recommended: open the repo in **VS Code** and use the integrated terminal.

### 1) Create a virtual environment + install dependencies

**Windows (cmd):**
<pre>
cd &lt;ROOT&gt;\03_sim_A\py
python -m venv &lt;ROOT&gt;\.venv
&lt;ROOT&gt;\.venv\Scripts\activate.bat
pip install -r &lt;ROOT&gt;\06_report\requirements_utf8.txt
</pre>

**macOS / Linux:**
<pre>
cd &lt;ROOT&gt;/03_sim_A/py
python3 -m venv &lt;ROOT&gt;/.venv
source &lt;ROOT&gt;/.venv/bin/activate
pip install -r &lt;ROOT&gt;/06_report/requirements_utf8.txt
</pre>

### 2) Run a smoke test

<pre>
python run_pipeline_A.py --run_id A_Smoke --preset RefPlus --scenarios Ref --rets 0,1,2 --seed_start 1 --n_seeds 1 --duration_s 60 --msg_rate_hz 10 --tx_mode mix --tx_k 6 --tx_k_cross 2
</pre>

Outputs go to:
<pre>
&lt;ROOT&gt;\05_results_A\runs\&lt;run_id&gt;\
  raw/
  tables/
  figures/
  run_commands.txt
</pre>
